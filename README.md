Production-grade data science analysis suite for blood donor retention, anomaly detection, parliamentary analytics, macroeconomic nowcasting, and social welfare assessment.

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/YOUR_USERNAME/bnm-assessment.git
cd bnm-assessment

# Install
pip install -r requirements.txt
npm install

# Run
python src/analysis/part_1a_retention.py
python src/analysis/part_1b_outlier.py
python src/analysis/part_2_hansards.py
python src/analysis/part_3_nowcasting.py
python src/analysis/part_4_sara.py

# Test
pytest tests/

# Deploy
npm run build
```

## 📊 What's Included

- **Part 1a:** Blood Donor Retention (Survival Analysis, Demographics)
- **Part 1b:** Outlier Detection (Multi-level, Ensemble Methods)
- **Part 2:** Parliamentary Hansards (NLP, Sentiment Analysis)
- **Part 3:** Macroeconomic Nowcasting (Bridge Equations, VAR Models)
- **Part 4:** SARA Assessment (Food Basket Optimization)
